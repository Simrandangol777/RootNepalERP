# Root Nepal Handicrafts ERP System  
**Smart Inventory & Restock Management for Traditional Businesses (FYP 2025/26)**

---

## 📖 Overview
**Root Nepal Handicrafts ERP** is a full-stack ERP system developed as a **Final Year Project** based on a **real family-owned handicraft business in Nepal**.  
It helps manage products, stock, purchases, sales, and reporting—while introducing an ML-powered **Smart Restock Suggestion** feature as the project’s **USP**.

---

## 🎯 Project Motivation
Small and medium handicraft businesses in Nepal still rely heavily on:
- Paper-based inventory tracking
- Manual sales records
- Spreadsheet-based restocking decisions

These methods often lead to:
- Overstocking or stock shortages
- Revenue leakage
- Poor visibility into sales trends
- Inefficient decision-making

This ERP system is designed to solve these real-world problems using modern web technologies and applied machine learning.

---

## ⭐ Unique Selling Point (USP): Smart Restock Suggestion
The system uses **historical sales + seasonal trends + current stock + product demand trends** to recommend **how much to reorder** for each product.  
This supports **data-driven inventory decisions** and reduces **stock-outs** and **overstocking**.
Using a trained **Machine Learning model**, the system predicts the optimal quantity of products to reorder, helping business owners:
-Reduce excess inventory
- Avoid high-demand stock shortages
- Improve cash flow efficiency
- Make accurate, data-driven restocking decisions
